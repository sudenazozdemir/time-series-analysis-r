# time-series-analysis-r
USD/TRY Time Series Analysis using R
## Project Contents
- Data collection from Yahoo Finance
- Stationarity analysis using Augmented Dickey-Fuller (ADF) Test
- First differencing transformation
- ACF and PACF analysis
- Structural break visualization
- ARIMA model selection
- Exchange rate forecasting

## Tools & Libraries
- R
- quantmod
- tseries
- forecast
- urca
- strucchange
- ggplot2

## Files Included
- R script file
- Analysis report
- Time series graphs and visualizations

## Key Findings
- The original USD/TRY series was found to be non-stationary.
- After first differencing, the series became stationary.
- A potential structural break was observed around December 2021.
- ARIMA modeling was used for forecasting future exchange rate behavior.
